O que são Heading ?

São elementos HTML que estruturam o conteúdo de uma página da web, indicando os títulos e subtítulos. São representadas pelas tags h1, h2, h3, h4, h5 e h6 elas também ajudam a

  • Facilitar a leitura
  • Ajudar os mecanismos de busca
  • Proporcionar uma experiência de navegação mais acessível

O que são listas ?

Voltar

São estruturas que organizam itens de forma sequencial ou categórica, são úteis para agrupar informações relacionadas, como etapas de um processo, itens de uma receita, entre outros

  • Lista ordenada:
    representa uma lista de itens ordenados, com uma contagem que os precede, como numerais, letras, algarismos romanos, ou símbolos. A lista ordenada é delimitada pelas etiquetas OL
  • Lista desordenada:
    representa uma lista de itens sem ordem rígida, com marcadores antes de cada elemento. A lista desordenada é delimitada pelo elemento UL

O que são links ?

Voltar

Também conhecidos como hiperlinks, são referências que permitem aos usuários navegar entre páginas da web ou documentos, São uma das características mais importantes da web e são a base para a interconectividade entre os sites, podem ser definidos pela a href=""

  • Navegar entre páginas
  • Vincular documentos
  • Indicar a relação entre páginas para os mecanismos de pesquisa

O atributo href é um componente dos links HTML que indica a relação entre páginas para os mecanismos de pesquisa. Ele contém dois elementos: a URL (o link real) e o texto ou objeto clicável que o usuário vê na página (conhecido como “texto âncora”).
